실습 19번의 기능을 그대로 살리기 위해 바뀐 명령어가 많습니다.
1/2 : 조명 켜기/끄기 ㅇ
c/C : 조명 색 변경 ㅇ
r/R : '조명'의 위치를 양/음 방향으로 회전 ㅇ
s/S : '모든' 움직임 정지 ㅇ

z/Z : 카메라가 z축 양/음 방향으로 이동
x/X : 카메라가 x축 양/음 방향으로 이동
y/Y : 카메라가 y축에 대하여 회전(자전)
a/A : 화면의 중심의 y축에 대하여 카메라가 회전(중심에 대하여 공전)

b/B : 크레인의 아래 몸체가 z축 방향으로 양/음 방향으로 이동한다.
m/M : 크레인의 중앙 몸체가 y축에 대하여 양 방향으로 회전한다.
n/N : 크레인의 중앙 몸체가 y축에 대하여 음 방향으로 회전한다.
t/T : 크레인의 맨 위 2개의 팔이 x축에 대하여 양/음 방향으로 서로 반대방향으로 회전한다.
