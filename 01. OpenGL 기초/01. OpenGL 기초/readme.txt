실습 1

소문자 입력시에도 작동하도록 하였습니다.

R : 빨간색으로 바뀝니다
G : 초록색으로 바뀝니다
B : 파랑색으로 바뀝니다
A : 0.0~1.0 사이의 값 중 하나로 바뀝니다. 경우의 수는 11^3이 되겠네요.
W : 흰색으로 바꿉니다.
K : 검정색으로 바꿉니다.
T : 0.2초 간격으로 A 명령어를 실행합니다.
S : T 명령어를 종료합니다.
Q : 종료합니다.

//////////////////////////////////////////////////////////////////////////////////////////
실습 2

초기 사각형 및 배경 설정값은 랜덤입니다. 0.0 ~ 1.0의 값 중 정해집니다.
사각형 내부 클릭하면 사각형 색 바뀌고 외부 클릭하면 배경 색 바뀝니다. 

추가기능으로 윈도우 크기를 임의의 크기로 변경해도 작동하도록 제작하였습니다.
또, 마우스 오른쪽 버튼 누르면 종료하게 했습니다.

//////////////////////////////////////////////////////////////////////////////////////////
실습 3

마우스 버튼 클릭한 채로 드래그 하면 사각형의 위치가 이동합니다.
사각형은 윈도우 밖으로 나갈 수 없게 했습니다.

윈도우 크기를 임의로 변경해도 작동하도록 제작하였습니다.
마우스 오른쪽 버튼 누르면 종료하게 했습니다.

//////////////////////////////////////////////////////////////////////////////////////////
실습 4

사각형의 생성을 환형 연결 리스트로 구현해 보았습니다. 

a :	사각형들이 대각선으로 이동합니다. 초기 이동 방향은 랜덤입니다.
	벽에 부딪히면 부딪힌 방향과 반대 방향으로 이동합니다 (완전탄성충돌)
c :	사각형이 위아래로 길게 늘어났다가 줄어들고, 
	좌우로 길게 들어났다가 줄어듦을 반복합니다.
	애니메이션이 진행 중에만 변경됩니다. 즉 s로 멈추면 사각형 크기 변경도 멈춥니다.
s :	사각형의 이동이 멈추고 변경도 멈춥니다.
m : 	사각형들이 초기 설정위치로 이동합니다.
r :	생성된 사각형들을 삭제합니다. 새로 입력 받을 수 있습니다.
q : 종료합니다.
