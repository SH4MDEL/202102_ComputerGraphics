장애물 높이를 낮게 만들었는데, 플레이어가 뛰어 넘어갈 수 있도록 구현하기 위함입니다.


i/j/k/l : 각각 앞, 좌, 뒤, 우로 이동합니다. 플레이어가 바라보는 방향 기반이 아니라
로봇의 절대 위치 기반으로 움직입니다.
즉 카메라를 돌려도 로봇의 이동은 변하지 않습니다.

w/a/s/d/ : 카메라가 앞, 좌, 뒤, 우로 이동합니다.

c : 카메라가 y축 양 방향으로 회전합니다.
C : 카메라가 y축 음 방향으로 회전합니다.

SPACE : 점프합니다. 장애물을 넘어갈 수 있습니다.

t/T : 조명이 꺼져 있다면 켜고, 켜져 있다면 끕니다.

ESCAPE : 모든 값을 초기화합니다.

q/Q : 종료합니다.
